## Java 8 lambdas

Runnable

With Java 7

class Runnable {
  public void run() {
    System.out.println("Hello World!");
  }
}

With Java 8 Lambda

() -> System.out.println("Hello World!");

Example: Creating an anonymous class

class Thread {
  public Thread(Runnable r) { ... }
}
 
new Thread(new Runnable {
  @Override
  public void run() {
    System.out.println("Hello World!");
  }
})

The λ idea
